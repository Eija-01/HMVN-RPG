====================

[ INDONESIA ]
====================

Engine game RPG sederhana berbasis HTML, CSS, dan JavaScript.

FITUR
------
✔ Movement diagonal unik
✔ Animasi walk & idle character
✔ Sistem pemilihan map
✔ Slider posisi map
✔ UI minimize
✔ Asset berbasis GitHub Repository
✔ Bisa dimainkan langsung di browser
✔ Tanpa library tambahan

--------
⬆️ = bergerak kanan atas ↗
⬇️ = bergerak kiri bawah ↙
⬅️ = bergerak kiri atas ↖
